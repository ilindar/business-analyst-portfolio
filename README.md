## О кейсе

**Проблема:**  
В спортивном клубе звонки от клиентов поступали на личный телефон тренера, информация не фиксировалась, конверсия лидов была низкой.

**Решение:**  
Разработана и внедрена новая система учёта:  
- Куплен рабочий телефон + CRM (1С:Фитнес-клуб, с временным переходом на «Тренерскую online»).  
- Описаны скрипты для администратора, путь клиента, регламенты работы.  
- В результате: обработка 90% заявок за 24 часа, конверсия лидов выросла до 70%.

**Документация включает:**  
- Бизнес-цели и критерии успеха  
- Бизнес-правила  
- User stories и роли  
- Функциональные требования (FR) с приоритизацией (метод взвешивания)  
- Use cases, BPMN-диаграммы  
- Макеты интерфейса (Miro)  
- Допущения, ограничения и бэклог развития
